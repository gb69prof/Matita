# Lavagna iPad (Web App)

## Come usarla
1. Apri `index.html` in Safari (iPad) o browser.
2. (Consigliato) Condividi → **Aggiungi a Home** per usarla a schermo pieno.
3. Penna/Gomma, colore, spessore, sfondo e griglia dal pannello.

## Salvataggi
- **Salva immagine (PNG)**: esporta la lavagna come immagine.
- **Salva progetto**: salva un file `.lavagna.json` (include snapshot PNG + impostazioni).
- **Apri progetto**: ricarica un progetto salvato.

### Nota su cartelle / iPad
Se il browser supporta il selettore file moderno (File System Access API), puoi scegliere la cartella.
Altrimenti il file verrà scaricato in “Download” e poi puoi spostarlo in Files.

## Avvio locale
- Basta aprire `index.html`.
